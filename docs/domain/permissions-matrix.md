## 공통 원칙

- 승인자는 화면에서 이름을 선택하지 않고 로그인 사용자 ID로 기록한다.
- 조직 경계를 서버에서 검증한다.
- 후원자 API 응답에는 대상자의 직접 식별정보가 없어야 한다.
- 권한 거절도 감사·보안 로그에 남긴다.
