# 1차 모바일웹 범위

## 포함

### 후원자
- 조직 계정 로그인
- 후원 임시 저장·사진 최대 8장·검토 요청
- 보완·재제출
- 제공 가능 수량·기한·전달 방식 관리
- 매칭 제공 확약·거절
- 픽업 준비와 완료 기록 확인

### 협의체 위원
- 비식별 케이스 등록·동의 근거 확인
- 운영 보완 요청 대응·재제출
- 승인된 케이스 자동 추천
- 연결 대기 후원 탐색·상세·케이스 연결
- 매칭 신청·수정·취소
- 전달 진행 확인

### 운영자·승인자
- 후원·케이스·매칭 검토
- 승인·보완·반려·종결
- 로그인 사용자 본인 기준 공동 승인
- 재고 예약·해제
- 배송·부분 완료·실패·취소·재배송
- 조직·사용자·감사 로그·비식별 보고

### 공통
- 로그인·조직·역할 권한
- 영구 저장·알림·감사 로그
- PWA·모바일 접근성
- HTTPS·백업·모니터링

## 제외
- 대상자 계정
- 후원자와 대상자 직접 채팅
- 결제·현금 후원
- 실시간 GPS 추적
- 앱스토어 배포
- 민감정보 오프라인 캐시
- AI 자동 승인·거절

## 1차 완료 기준

후원 등록부터 완료 보고까지 정상·보완·반려·취소·부분 완료 경로가 서버 상태 전이로 닫히고, 새로고침·재로그인 뒤에도 데이터가 유지되어야 한다.
